Govan's waste management system

This is a basic website for Govan's waste management system. This is just a simulation not an actual implimentation, it simulates a waste management system that is used to track Govan's bins from several locations. The system recieves the statuses of the bins from a sensor system and displays it on the web app. The system also notifies the local municipality when bins are full.
